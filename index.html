Claude Code 環境構築＋セキュリティ
2日目のゴール：
Claudeデスクトップで動かせる / スプシ書き込み / Git管理 / セキュリティ対策の4つを実施する
  • Claudeデスクトップ 環境構築ハンズオン
    • 操作モード5種類（プランモードがおすすめ）
    • CLAUDE.md の作り方・設定方針
    • モデル選択（Sonnet 4.6が日常使いの主役）
  • ハンズオン1：スプシ書き込み環境を作る → 「スプシ書き込み環境」タブ
  • ハンズオン2：体験レポートページを作る → 「体験レポートページ」タブ
  • ハンズオン3：Vercel に公開してみよう → 「Vercel公開」タブ
  • ハンズオン4：Git & GitHub を設定しよう → 「Git&GitHub」タブ
  • よいアウトプットに必要な「設計」の考え方
    • ゴール・ターゲット・素材の3点セットを用意する
    • 設計なし → 何度もやり直し → 時間もトークンもムダ
  • AIと安全に仕事するために（ セキュリティ対策タブ）
    • 怖いこと1：PCのファイルが消える・書き換えられる
    • 怖いこと2：APIキー（鍵）が漏洩する
    • 怖いこと3：外部から操られる（プロンプトインジェクション）
